How It Works

Pattern: hash ‚Üí receipt ‚Üí link ‚Üí events ‚Üí verify

Try it in 30 seconds

# 1) Hash your message locally (or hash a PDF / attachment if you prefer)
# 2) Create a receipt (server stores the receipt, not the message)
curl -X POST "https://ackzero.io/v1/proofs" \
  -H "Content-Type: application/json" \
  -H "Accept: application/json" \
  -d '{"hash":"<sha256_of_message_or_attachment>","kind":"email","meta":{"subject":"Termination notice"}}'

# Response includes:
# - id (stable receipt id)
# - token (non-guessable link token)
# - claim_url: https://ackzero.io/p/<token>
# - receipt_url: https://ackzero.io/v/<id>
# Insert claim_url into your email. Keep receipt_url for verification.

1) Hash locally

const buf = new TextEncoder().encode(messageText);
const digest = await crypto.subtle.digest("SHA-256", buf);
const hash = [...new Uint8Array(digest)]
  .map(b => b.toString(16).padStart(2, "0"))
  .join("");

2) Create the receipt

const r = await fetch("/v1/proofs", {
  method: "POST",
  headers: { "Content-Type":"application/json", "Accept":"application/json" },
  body: JSON.stringify({
    hash,
    kind: "email",
    meta: { subject: "Example" }
  })
});
const receipt = await r.json();

3) Verify anytime

// Public receipt JSON (signed)
const v = await fetch("/v/" + receipt.id, {
  headers: { "Accept":"application/json" }
});
const proof = await v.json();

Output: a receipt JSON with id, hash, timestamps, event trail, and a signature ‚Äî enough for independent verification.
